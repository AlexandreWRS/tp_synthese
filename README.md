# Projet réalité augmenter 
## Concept
### Rentrer dans le monde des samouraïs de l'apparition à la fin de cete ère. 
1. Projection immersive 360°
2. Rencontre des samourais reconnus( les légendes)
4. Mise en contexte socio-politique du Japon médiéval
5. Narration historique accompagnée de musique traditionnelle
## Comment ca marca
L’utilisateur se déplace librement dans plusieurs environnements reconstitués 
1. un village rural du XIIe siècle
2. un dojo d’entraînement
3. un château japonais (shiro) à l’époque Sengoku
4. le champ de bataille
5. une rue d’Edo à l’époque Tokugawa
6. Chaque lieu déclenche des capsules historiques audio-visuelles (narration, animations, hologrammes).
## Pour qui?
les personnes qui sont passionnés ou bien qui s'intéressent à l'histoire du japon.
